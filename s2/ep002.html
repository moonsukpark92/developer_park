GPU 가격을 알아봤다. 파워서플라이는 국내에 안 팔아서 알리익스프레스를 뒤졌다. 메모리 호환 모델도 찾아봤다. 하나하나 부품을 맞춰가는 과정.

그런데 이건 단순한 서버 구매가 아니었다. 이건 — 생각이 "땡" 하는 순간이었다.

Chapter 05

자체 LLM +
Claude 하이브리드

그래서 만들려는 것.

단순한 작업 — 일상적인 문서 정리, 기본 응답, 반복 업무 — 이런 건 Ollama 같은 자체 LLM으로 돌린다. 오픈소스 모델로 충분하다.

복잡한 사고 — 재무 분석, 규정 컨설팅, 전략적 판단 — 이런 건 Claude API를 쓴다. 진짜 두뇌가 필요한 일에만.
