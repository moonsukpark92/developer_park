01 · 롤모델

목표는
Genspark이었다

Genspark의 슈퍼 에이전트에는 대화 모드가 있다. 직접 써봤다. 대화가 자연스럽고 매끄럽게 잘 된다. 단순히 말하면 받아쓰는 수준이 아니라, 실질적인 AI 에이전트와 대화하는 느낌이었다.

목표가 생겼다. 저 수준의 대화가 되는 자비스를 만들어야겠다. 단순히 음성을 인식해서 명령을 전달하는 것이 아니라, 진짜 대화가 되는 형태. 그게 JARVIS CC 프로젝트의 롤모델이 됐다.

02 · 생각의 확장

말 한마디로
코딩이 되면 어떨까

처음 아이디어는 단순했다. VS Code에서 직접 타이핑하는 대신 말로 시키면 안 될까. 그게 점점 커졌다.

1단계 · 출발

VS Code에서 타이핑할 내용을 말로 시킬 수 없을까?

2단계 · 확장

VS Code가 아니라 Claude Code 자체에게 말로 대화하게 만들어버리면 되는 거 아닌가?

3단계 · 결정

그렇다면 프로그램으로 완전히 만들어보자. 구글링하고, 유튜브 뒤지고, GitHub 탐색 시작.

4단계 · 탄생

GitHub에서 유사 프로젝트를 찾아 전부 클론해 다운로드했다. EP006 방법론대로 소스코드 분석, 계획서 작성, 현재 프로젝트에 반영. JARVIS CC 프로젝트가 시작됐다.

계획서는 Genspark과 Claude를 활용해 정리했다. 차량 운행 중 녹음한 파일로 메모하면서 생각을 다듬었다. 그 결과물이 오늘 아침 빌드 지시로 이어졌다.

03 · 프로젝트 스펙

JARVIS CC —
무엇을 만드는가

JARVIS CC. Jarvis for Claude Code. 개인 PC와 노트북에서 단독으로 작동하는 음성 대화형 AI 비서 프로그램이다.

JARVIS CC
Jarvis for Claude Code · 개인 AI 비서 · 자체 개발 중
핵심 기능
말로 Claude Code와 대화. 타이핑 없이 음성으로 지시하고 결과를 받는다.
운영 환경
개인 PC / 노트북 단독 모듈. 별도 서버 없이 로컬에서 완전히 작동한다.
확장 목표
VS Code 등 일상 업무 프로그램까지 음성으로 제어. 모든 작업을 말 한마디로.
메모리 · 맥락
대화 내용 전체를 기억하고 정리. 과거 맥락까지 이해해 더 정확한 지시가 가능해진다.
개발 방식
GitHub 클론 + Anthropic 공식 문서 + 인터넷 서칭 → 소스코드 분석 → 계획서 → 빌드.
최종 비전
완성되면 제품화. 같은 환경의 누구든 쓸 수 있는 독립 모듈로 만든다.
04 · 오늘 아침

빌딩을
시작하라

오늘 아침, Claude Code에게 빌드를 시작하라고 지시했다. 실제로 초창기 모델을 한두 시간 만에 만들어 바로 테스트해봤다. 결과는 냉정했다.

✓  초창기 모델 빌드 완료 (약 1~2시간)
✓  테스트 진행
⚠  한글 인식 불량 — 음성 인식 정확도 낮음
⚠  반응 속도 느림 — 실사용 수준 미달
⚠  명령 오작동 — 의도한 대로 처리 안 됨
○  개선 작업 진행 중
○  완료 후 GitHub 푸시 예정
// 단순히 말하고 받아쓰는 게 아니라, 진짜 대화가 되는 자비스가 목표다.

이 문제들을 해결하기 위해 Anthropic API와 OpenAI API 키를 두 개 모두 세팅했다. 각 월 100달러씩, 예산 범위 안에서 집행 가능하도록 설정하고 키 두 개를 주고 나왔다. 두 API를 함께 활용해 한글 인식과 응답 품질 양쪽을 잡는 것이 다음 목표다.

과정은 AI가 처리한다. 중요한 것은 내가 무엇을 만들려고 하는지, 그 방향과 아이디어였다. 그것만 명확하면 AI가 나머지를 해낸다. 그게 지금 AI 시대다.

05 · 생각의 확장

AI 시대는
생각의 크기가 결정한다

JARVIS CC 프로젝트를 진행하면서 이런 생각이 들었다. 지금 AI 시대는 뭔가를 만들려는 아이디어가 중요한 것이지, 그 과정은 별로 중요하지 않다. 내 아이디어를 현실로 바꾸는 것. 그게 지금 AI 시대의 본질이다.

AI 시대는 있는 모든 시간을 쪼개고
내 생각을 얼마나 확장하느냐에 따라
엄청난 변화가 될 것 같습니다.

JARVIS CC가 완성되어 개인 PC에서 단독 모듈로 가동된다면 제품이 된다. VS Code 같은 일상 업무 프로그램에도 이식 가능한 수준으로 발전하고, 대화 내용을 메모리해서 과거 맥락까지 이해한다면 — 그것은 진짜 자비스가 된다.

이 시대에 진정 필요한 것이 무엇인가.
JARVIS CC 프로젝트가 그 질문을
다시 생각하게 만들었다.
오늘 밤, 잘 되어 있기를 바란다.
