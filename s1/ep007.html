Chapter One

재무담당자와
문실장이 같은 방에서
대화한다

텔레그램에 재무회계 전용 채팅방을 만들었다. 문실장을 초대하고, 조*진 프로를 초대했다. 이제 재무담당자가 직접 문실장에게 질문하고 지시한다.

직원과 AI가 같은 공간에서 일한다. 대표는 방향을 제시하고, 문실장이 분석하고, 직원이 실행한다. 이게 새로운 조직 구조다. 인사팀이 없어도, 법무팀이 없어도, 재무팀이 없어도 된다.

02

Chapter Two

이력서를 보다
문득 이런 생각이.
문실장이 먼저 검토하면?

영업부 지원자 이력서가 들어왔다. 문득 이런 생각이 들었다. "이 이력서를 문실장이 분석하고 의문 사항을 추출하고 사전 질문 메일을 보내면?" 그냥 생각이 현실이 됐다.

이력서를 텔레그램으로 올렸다. "인사 전문가 입장에서 분석해줘." 7가지 사전 질문이 담긴 이메일이 지원자에게 자동 발송됐다. 재직 기간 중복, 단기 퇴직 사유, 실적 기여도 검증, EPC 인증서 미첨부까지. 면접관은 더 중요한 판단에만 집중한다.

같은 날 단양 공장 사무직 지원자, 신입 지원자도 왔다. 3명 모두 문실장이 분석하고 맞춤 질문지를 만들었다. 구글 드라이브에 채용관리 폴더가 자동 생성됐다.

03

Chapter Three

모든 직원이
자신만의 문실장을
가진다면

이동 중에 ChatGPT에게 물었다. "노후 노트북에 리눅스 깔고 Claude Code 채널 기능으로 직원 전용 업무 봇 만들 수 있어?" 된단다. 단양에 도착하자마자 Claude Cowork에게 만들어달라고 시켰다.

agent-usb-installer. USB 하나로 자동 설치. 부팅하면 모든 것이 준비된다. 회사 노후 노트북에 꽂으면 Ubuntu 22.04 + Claude Code + 텔레그램 봇이 자동 설치된다. 문실장이 1달간 교육받은 내용이 파일로 이전돼 즉시 인식한다.
